# 交大学生成绩管理系统

这是一个独立的模拟业务网站，用于课程项目中的业务系统场景展示。当前作为独立子项目放在 IDS/IPS 项目目录下，但不依赖 IDS/IPS 的前端路由或后端应用。

## 功能

- 学生用户登录后只能查看自己的成绩。
- 教师用户登录后可以查看全部学生成绩。
- 教师用户可以登记新成绩，也可以按学号和课程修改已有成绩。
- 后端会校验登录态、角色权限、学生是否存在、课程名称和 0 到 100 的整数成绩。

## 运行

从仓库根目录进入独立子项目后启动：

```bash
cd "交大学生成绩管理系统"
python app.py
```

默认访问地址：

```text
http://127.0.0.1:8000
```
